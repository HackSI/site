The HackSI.me website
=====================

The hacksi.me website

Development
-----------

   * Install [Node.js](http://nodejs.org/)
   * Clone this repo
   * `npm install`
   * `npm start`
   * Visit `http://127.0.0.1:8900/`

Testing & Linting
-----------------

`npm test`

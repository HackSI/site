The HackSI.me website
=====================

The hacksi.me website

Development
-----------

   * Install [Node.js](http://nodejs.org/)
   * Clone this repo
   * `cd site`
   * `npm install`
   * `npm start`
   * Visit `http://127.0.0.1:8900/`

Testing & Linting
-----------------

`npm test`

Production
----------

`npm -g install grunt-cli`

   * `cd site`
   * `grunt`
   * `git commit -am "CSS files"`
